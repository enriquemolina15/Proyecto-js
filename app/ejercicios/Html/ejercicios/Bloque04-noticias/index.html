¿Qué es HTML5?

HTML5 es la versión más reciente del lenguaje de marcado estándar para la creación de páginas web.

¿Por qué es importante?

Permite una estructura más clara del contenido, mejora la accesibilidad y ofrece nuevas etiquetas semánticas.
