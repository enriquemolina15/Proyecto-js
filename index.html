Introduccion

Este proyecto esta realizado con HTML,CSS y JAVASCRIPT.El objetivo es aplicar todo lo aprendido,ademas de investigar y ampliar los conocimientos adquiridos.

Funciones implementadas
  1. Conversion decimal a binario => usando logica personalizada.
  2. Mini calculadora => usando operaciones basicas.
  3. Generador de numeros aleatorios.
  4. Uso de Switch-case para mostrar diferentes resultados segun opciones.
  5. Uso de expresiones regulares.
  6. Proyecto de una calculadora cientifica
Funciones implementadas
  1. Fetch a JSON Placeholder para mostrar informacion.
  2. Promesas con then y async/await
  3. Almacenamiento en cookies,localStorage,sessionStorage,indexdDB
Contenido extra

Para finalizar e implementar este proyecto e incluido un login,que mantiene la sesion usando localStorage y esta landing page.
